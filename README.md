# Embrained.ai

The public-facing website for **Embrained, LLC**, a neurorobotics company based in Ann Arbor, Michigan.

**Live Site:** [https://embrained.ai](https://embrained.ai)

### Overview
This repository hosts the static landing page for the Embrained "Soft Launch" (Q1 2026). It is designed to be lightweight, privacy-focused, and agent-friendly.

### Tech Stack
* **Core:** HTML5, CSS3 (No frameworks)
* **Hosting:** GitHub Pages
* **DNS/Security:** Cloudflare
